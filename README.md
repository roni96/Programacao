**Projeto - Port Scan** <br>
Realiza o scanner de portas TCP. <br>

Para executar o script, na hora da sua chamada passa como parametro a rede e a porta para realizar o scan  

**Sintaxe:**  
`python .\port-scan.py 192.168.10.0/24 80`
